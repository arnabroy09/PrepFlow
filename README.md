## ✨ Core Features

### 📝 1. Multiple Scratchpads (Notes)
* Create unlimited quick notes and scratchpads.
* One-click "Copy to Clipboard" functionality.
* **Export Options:** Download all your notes beautifully formatted as a **PDF** or standard **.txt** file.

### 🗺️ 2. Syllabus Kanban Board
* A drag-and-drop interactive board for syllabus tracking.
* Move topics seamlessly between **To Study ➔ Revising ➔ Mastered**.
* Color-coded tags for subjects to prevent study overwhelm.

### 🗂️ 3. Study Resource Manager
* Maintain a structured database of important YouTube links, articles, and PDFs.
* Auto-populating subject filter.
* **Export Options:** Download your entire database as an Excel-compatible **.CSV** file.

### ✅ 4. Priority Task List
* Daily To-Do list with a visual progress bar.
* **Top Priorities:** Star specific tasks to mark them as high priority.
* One-click clear completed tasks.

### 🧠 5. Active Recall Flashcards
* Create custom Question & Answer decks.
* Premium **3D Glassmorphism flip animations** to test your memory.
* Built-in card counter and easy navigation.

### ⏱️ 6. Focus Timer & Analytics
* Built-in Pomodoro-style timer (15m, 25m, 50m presets).
* **Session Logger:** Automatically tracks your focused study time.
* **Visual Analytics:** A dynamic, CSS-based 7-day bar chart showing your daily study hours.

### 🎯 7. Multiple Exam Countdowns
* Add unlimited target exams (e.g., UPSC, SAT, College Finals).
* Live horizontal slider displaying exactly how many days are left for each goal.

### 🌙 Premium UI & Experience
* **Deep Dark Mode:** Eye-friendly slate/zinc dark theme that saves your preference.
* **Daily Streaks:** Automatically tracks consecutive days you open the app to build consistency.
* Fully responsive (Mobile, Tablet, and Desktop friendly).

## 🔒 Privacy & Data Security (100% Offline)
**No servers. No databases. No logins.** 
PrepFlow runs entirely in your browser using Vanilla JavaScript and `localStorage`. 
* **Your data never leaves your device.** 
* **Backup & Restore:** You can download your entire workspace as a secure `.json` file and upload it later to restore your data on any device.

---

## 🛠️ Tech Stack
PrepFlow is built with absolute simplicity and speed in mind. It is a **Single-File Web Application**.
* **HTML5**
* **CSS3** (Custom Properties, Flexbox/Grid, Glassmorphism, 3D Transforms)
* **Vanilla JavaScript** (ES6+)
* *External Libraries:* FontAwesome (Icons), HTML2PDF.js (For PDF Export)
